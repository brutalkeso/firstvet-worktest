# Development environment
Step by step on how to setup the development environment

## Install nvm
https://github.com/nvm-sh/nvm

## Install and use node 20.5.1
nvm install 20.5.1
nvm use 20.5.1

# Running app
npm install
npm run start

If your running on device, duplicate the .env file and name it .env.local, 
changing the url to your computers ip.
